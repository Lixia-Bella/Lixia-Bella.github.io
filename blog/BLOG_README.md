## 主题风格

色系改造 - 马卡龙色系

用5种柔和的马卡龙色替换了原来的阳光橙+柠檬黄：

马卡龙粉 #F2A6B3 - 主色调
薄荷绿 #A8D8C8 - 辅助色
淡紫色 #C3B1E1 - 辅助色，增加层次
奶油黄 #F9E4B7 - 点缀色
天蓝色 #A7C7E7 - 点缀色
设计改进内容

背景设计 - 使用 body::before 添加了多层柔和的径向渐变光晕，形成微妙的几何图案效果

卡片层次 - 使用双层阴影 shadow-card 组合粉色和紫色阴影，悬停时阴影加深并微微上浮，增强层次感

动效细节：

卡片悬停上浮 translateY(-2px) + 阴影加深
爱好卡片悬停图标放大 scale(1.15)
联系卡片悬停上浮+缩放 translateY(-5px) scale(1.03)
使用 cubic-bezier 缓动函数让动画更自然
排版优化 - 增大行高到 1.7，调整字体层级和间距，标题使用深紫色 --text-heading

色彩搭配 - 每张爱好卡片、技能标签、联系卡片都使用不同的马卡龙色，每个章节顶部有不同颜色的装饰条，丰富但不花哨
